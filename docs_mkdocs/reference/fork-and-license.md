# Fork, Upstream, and License

This repository should be understood as a maintained, holography-oriented fork
built on top of the original `PADO` project.

## Upstream Relationship

- upstream core project: [`PADO`](https://github.com/shwbaek/pado)
- original framework affiliation:
  [`POSTECH Computer Graphics Lab`](https://sites.google.com/view/shbaek/home)
- maintained repository direction here: `PADO Hologram`

The core package path remains `pado` for compatibility with the original layout.

## Fork Identity

This repository keeps compatibility where possible, but it is being developed as
a more explicit holography framework layer on top of the original optics core.

That includes:

- `pado.display`
- `pado_hologram`
- updated documentation
- additional regression tests
- repository-level workflow and packaging changes

## License

The repository remains under the MIT license inherited from the original PADO
project.

In practical terms, the current license permits:

- use
- modification
- distribution
- sublicensing

provided that the required copyright and license notices are preserved.

This makes a maintained extension such as `PADO Hologram` compatible with the
current repository license structure. This is a technical reading of the current
license text, not formal legal advice.
