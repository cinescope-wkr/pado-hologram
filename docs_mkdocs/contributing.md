# Contributing

Contributors are welcome.

If you have experience with related projects such as
[`holotorch`](https://github.com/facebookresearch/holotorch), your perspective is especially valuable.

Good areas to help with:

- hologram-generation algorithms
- SLM and display models
- multi-plane CGH workflows
- Hydra configs and experiment tooling
- tests, examples, and documentation

See the repository-level contribution guide on GitHub:
[`CONTRIBUTING.md`](https://github.com/cinescope-wkr/pado-hologram/blob/main/CONTRIBUTING.md)
